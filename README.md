# Exchange

A currency exchange rate calculator built with UIKit for learning purposes.

## Features

- Convert USD to other currencies and vice versa
- Two input fields with real-time conversion
- Currency picker via bottom sheet
- Swap currencies with one tap

## Tech Stack

- UIKit (programmatic UI)
- MVVM architecture
- Swift 5

## Requirements

- iOS 15+
- Xcode 15+

## Getting Started

1. Clone the repository
2. Open `Exchange.xcodeproj` in Xcode
3. Build and run (⌘R)
